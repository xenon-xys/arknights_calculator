# 明日方舟抽数计算器

一个用于计算《明日方舟》从指定日期开始到截止日期可获得的合成玉、至纯源石、寻访凭证数量及对应抽数的网页工具。

## 功能特性

### 📊 资源计算
- **合成玉**：日常稳定资源 + 限时活动资源 + 搓玉
- **至纯源石**：日常稳定资源 + 限时活动资源
- **寻访凭证**：日常稳定资源 + 限时活动资源 + 黄票商店兑换
- **抽数转换**：自动计算总抽数（600合成玉=1抽，3.33源石=1抽）

### 📅 日期处理
- 支持"当前时间"或"指定日期"选择开始日期
- 游戏日期转换：现实时间凌晨0:00-3:59按前一天计算
- 周计算：周日开始（日-六）
- 月计算：精确跨月计算

### 🎯 日常资源
- **每日资源**：任务、剿灭、月卡等
- **每周资源**：剿灭、周常任务、绿票商店等，支持"本周已获取"
- **每月资源**：签到、黄票商店、绿票商店等，支持"本月已领取"
- 一键全选/取消全选

### 🎉 限时活动
- 自定义添加/删除活动
- 活动类型：稳定 / 不确定
- 获取方式：单次领取 / 每日获得
- 日期筛选：只显示与计算日期范围有交集的活动
- 一键全选/取消全选
- 默认包含多个预设活动（夏活系列、红丝绒复刻等）

### 🏪 黄票商店
- 根据日期范围自动显示可兑换月份
- 每月可兑换38寻访凭证（需258高级资质凭证）
- 自定义数量兑换

### ⚙️ 搓玉功能
- **固源岩搓玉**：1固源岩+800龙门币=5合成玉
- **装置搓玉**：1装置+1000龙门币=10合成玉
- 自定义输入材料数量

### 📦 当前库存
- 输入当前合成玉、源石、寻访凭证数量
- 快捷数值按钮
- 汇总计算：库存 + 预期获取

### 🎨 界面功能
- 深色/浅色模式切换
- 响应式布局（支持电脑和手机）
- 资源图标显示

## 使用方法

### 本地运行
1. 克隆或下载项目
2. 使用浏览器打开 `index.html` 文件
3. 或使用本地服务器：
   ```bash
   python -m http.server 8000
   ```
4. 访问 `http://localhost:8000`

### 线上运行
- `https://xenon-xys.github.io/arknights_calculator/`

## 计算规则

### 资源转换
- 1至纯源石 = 180合成玉
- 600合成玉 = 1抽
- 1寻访凭证 = 1抽
- 3.33源石 ≈ 1抽

### 日期边界
- 游戏日期以凌晨4点为界
- 周开始为周日
- 每月17号签到领取寻访凭证

## 注意事项

1. 所有数据仅存储在本地浏览器中，不会上传到服务器
2. 限时活动数据会自动保存，刷新页面后不会丢失
3. 搓玉功能需要用户自行保证获取订单效率
4. 计算结果仅供参考，实际获取数量可能因游戏更新而变化
